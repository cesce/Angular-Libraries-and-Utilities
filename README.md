## Tables

### ngx-easy-table

The Easiest Angular Table. IVY compatible. 9.0kb gzipped! 60fps! Tree-shakeable. 54 features and growing!

<https://ngx-easy-table.eu/#/>
<https://github.com/ssuperczynski/ngx-easy-table>

### ngx-datatable

ngx-datatable is a Angular component for presenting large and complex data. It has all the features you would expect from any other table but in a light package with no external dependencies. The table was designed to be extremely flexible and light; it doesn't make any assumptions about your data or how you: filter, sort or page it.

It was built for modern browsers using TypeScript, CSS3 and HTML5 and Angular >=4.0.0. This is the sister project of the angular-data-table that is designed for Angular 1.x.

<https://swimlane.gitbook.io/ngx-datatable/>
<https://github.com/swimlane/ngx-datatable>

### ng2-table

Simple table extension with sorting, filtering, paging... for Angular2 apps

<https://valor-software.com/ng2-table/>

## Charts

### ng2-charts

<https://valor-software.com/ng2-charts/#pieChart>
<https://github.com/valor-software/ng2-charts>

### ngx-charts

Declarative Charting Framework for Angular!.

ngx-charts is unique because we don't merely wrap d3, nor any other chart engine for that matter. It is using Angular to render and animate the SVG elements with all of its binding and speed goodness, and uses d3 for the excellent math functions, scales, axis and shape generators. By having Angular do all of the rendering it opens us up to endless possibilities the Angular platform provides such as AoT, SSR, etc.

<https://swimlane.github.io/ngx-charts/#/ngx-charts/bar-vertical>
<https://github.com/swimlane/ngx-charts>
